My work in open-source

When I was checking the GitHub page of a package that I was using, I noticed that they had a few issues open. I started by adding and improving hash type detection in a tool called Name-That-Hash. It is a popular tool used to get the type of a hash. You give it a hash and it will try to find out the type. It also returns john and hashcat mode names for cracking that hash type.

Because I worked a lot with regex when contributing to Name-That-Hash, I was invited to help with a tool called PyWhat before it was even public. PyWhat is another tool from Bee, who is an amazing person and a creator of Name-That-Hash! PyWhat's job is to identify and return specific information from text, files or even whole folders. It can find IP addresses, emails, phone numbers, coordinates, Bitcoin wallet addresses and a bunch of more stuff.

AUR

As an active Arch user, I noticed that a few packages I was using were a few versions behind and without a maintainer (ophaned). The packages were in AUR which is short for Arch User Repository. There users can upload little scripts to download and install a specific version of a package. I adopted three packages that I was using and that were without a maintainer. I am also the maintainer of the python-name-that-hash package.

When new updates get released, I update the package and users that downloaded my package from AUR also get the update.

I am planning to upload all of the PKGBUILD scripts to GitHub.
