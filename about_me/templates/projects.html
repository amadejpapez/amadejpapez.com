My work in open-source

I started by adding and improving hash type detection in a tool Name-That-Hash. It is a popular tool used to get the type of a hash and john or hashcat mode names for cracking that type.

Because I worked a lot with regex when contributing to Name-That-Hash, I was invited to help with a tool called PyWhat before it was even public. PyWhat is another tool from Bee, who is an amazing person and also a creator of Name-That-Hash. PyWhat's job is to identify and return specific information from text, files or even whole folders. It can find IP addresses, emails, phone numbers, coordinates, Bitcoin wallet addresses and a bunch of more stuff.

AUR

As an Arch user, I noticed that a few packages I was using were outdated and without maintainer in the AUR. This is short for Arch User Repository, where users upload little scripts to download and install a specific version of a package - similar to an App Store. I adopted three packages that I was using and were without a maintainer. I am also the maintainer of the python-name-that-hash package.

I am planning to upload all of the PKGBUILD scripts to GitHub.
